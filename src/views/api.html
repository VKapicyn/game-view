HTTP API

Доступны следующие GET запросы:

https:hrhonor.ru/api/v1/{token}/{type}/status - текущий раунд
https:hrhonor.ru/api/v1/{token}/{type}/txs - все транзакции
https:hrhonor.ru/api/v1/{token}/{type}/credits - все кредиты
https:hrhonor.ru/api/v1/{token}/{type}/adverts - все объявления

где,
{type} - формат возвращаемых данных, доступны json и csv;
{token} - токен доступа, цена 5 у.е. на один раунд, покупка через представителя банка.
